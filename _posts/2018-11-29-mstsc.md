# Windows 远程桌面连接相关操作

## 一、打开方式

1. 开始 -> 所有程序 -> 附件 -> 远程桌面连接
2. Windows+R -> 输入 mstsc


## 二、隐藏顶部连接栏

1. 在进行远程连接前，点击`显示` -> 取消勾选`全屏时显示连接栏`
2. 退出全屏：<kbd>Ctrl</kbd>+<kbd>Alt</kbd>+<kbd>Pause</kbd>


## 三、调整远程桌面显示效果

1. ClearType字体：点击`体验` -> 勾选`字体平滑`
2. 主题桌面设置：点击`体验` -> 勾选`桌面背景`
